# MoYu
摸鱼不翻车
一款专为职场人打造的轻量级 Android 趣味 APP，结合「佛系敲木鱼解压」+「搞钱专注计时」双核心功能，用轻松的方式缓解职场焦虑，兼顾摸鱼与搞钱～

**🌟 项目简介**
“摸鱼续命机” 聚焦职场人 “解压摸鱼” 和 “专注工作” 的双重需求，通过「敲木鱼攒功德」缓解工作焦虑，「搞钱专注钟」记录工作时长，搭配趣味文案和单日数据统计，打造轻量化、无广告的职场解压小工具。

**📱 核心功能**
1. 账号体系（登录 / 注册）
自定义「摸鱼代号」和「续命密码」，本地存储账号信息，无需联网
多用户隔离：不同账号的木鱼计数、工作时长数据互不干扰
2. 佛系敲木鱼 - 功德攒攒乐
点击木鱼图片触发震动反馈，实时统计「今日功德」（单日敲击次数）
跨天自动重置计数，每敲 10 次触发趣味提示（如 “功德 + 1！今日摸鱼焦虑 - 10%”）
3. 搞钱专注钟 - 卷王修炼场
一键启动 / 暂停工作计时，实时展示当前专注时长（时：分: 秒）
统计单日累计工作时长，未达到 2 小时阈值时弹出趣味提醒（如 “还需卷 1 小时 20 分钟，快去敲木鱼续命～”）
达标后触发正向反馈（如 “今日搞钱时长达标！功德圆满，可摸鱼 1 小时～”）

**🛠️ 技术栈**
开发语言：Java
布局框架：LinearLayout
本地存储：SharedPreferences
核心工具：
Handler：实现工作计时的秒级刷新
SimpleDateFormat：单日数据的日期标记与重置
